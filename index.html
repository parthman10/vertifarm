⚙️ How It Works

📡 Sensor Data
Sensors continuously collect real-time data like moisture, temperature, and humidity to monitor plant health.
🧠 Analysis
The system analyzes collected data using smart logic to decide what action is needed for optimal growth.
⚡ Automation
Automation controls water pumps, lights, and fans automatically based on plant requirements.
📱 Monitoring
Users can monitor and control the system through a dashboard with live updates and alerts.

🌟 Key Features

💧 Irrigation
Automatic irrigation system ensures plants get the right amount of water at the right time.
📊 Monitoring
Real-time monitoring allows users to track plant conditions like moisture, temperature, and humidity.
💡 Lighting
Smart lighting system provides optimal light conditions for plant growth at all times.
🌡 Climate
Climate control maintains ideal temperature and humidity levels for healthy plant development.
